Ability Management Cube

Ability Insight - SubAbility of Analyze Ability

Allows users to see the Abilities any object, creature, or person has.

Ability Copy

Allows user to copy any Ability

All Abilities copied will be stored in Ability Storage

Ability Editor

Allows user to rewrite the Abilities name and meaning

Allows user to adjust mana cost

Allows user to adjust time dirtrations

Allows user to adjust volume amounts

Allows user to adjust percentage amounts

Ability Placement

Allows user to place the copied/edited Ability on to any object, creature, or person

Placement can be done creating a tattoo, transferring mana (Method), blood consumption, and/or sigil

Ability Program

With Ability "Ability Restrictions" the user can create a set of instructions and/or rules while using the Ability "Ability Editor"

Ability Restrictions

Rules, regulation, limits, requirements can be placed on the copied/edited Ability

Ability Evolution

When user is given permission to edit Abilities that belong to other objects, creatures, and/or person(s)

Must request permission over Ability Storage

Ability Memory of Knowledge

User keeps perfect memory of any Abilities copied

Keeps knowledge of original Ability

Keeps knowledge of edited Ability

Ability Storage

Stores the information of all analyzed Abilities so they can be recreated later.

Akashic Records

Omni-Magic of Formation

The user can scan and analyze any magic chant, magic circle, magic enchantment, magic Inscription, and magic rune that is the user's view.

Once the user has full understanding of the magic they would be able to recreate it within the user's view.

Magic Creation

User is able to create entirely new forms of magic or create magic itself, including crafting/designing their own rules, limitations, maximum potentials, taboos, and everything else necessary for the new form of magic they created, they can even remove the magic they crafted from others if they feel it's being misused.

Magic Unification

Users can unify different types of magic, combining their advantages and possibly creating stronger forms of magic.

Spell Manipulation

The user can create, shape and manipulate spells, formulae for producing specific magic effects. Such formulae are often a series of magic words and incantations, but they can also take the form of specific thoughts, gestures, rituals, etc.

Multiple Activation/Casting

The user can activate/cast the same and/or different skills/abilities multiple times simultaneously.

Multiple Layering

The user can create, shape and manipulate layers, shaping, adding, removing and reinforcing them from different kinds of things.

Universal Replication

User can perfectly replicate themselves and/or targets.

Target can be inorganic, organic, physical, or non-physical item, Skill/Ability, matter and/or energy.

All replicated items would be created out of the user's mana.

Universal Transfer

The user can transfer Skills/Abilities, Mana/Energy, Life Force, Status, etc to anyone or things.

Arcane Eyes of Ultra Alchemy

Alchemical Entity Creation

User is able to create entities through alchemy, using the process of alchemy and using alchemical properties and materials to create living beings that the user can command and change the properties of using their control over the process of alchemy.

Can grant these beings varying levels of detail (simple to intricate), independence (controlled, programmed, semi-independent or autonomous) and existence (momentary to permanent) and delete the entities once they are done with them.

Artisan Mode

User can craft items/objects that may be functional or strictly decorative out of any/all materials or substances including non-physical and abstract materials.

Such as fire, clouds, water, bubbles, smoke, air molecules, moonlight, lava, sound, music, and even emotions like joy, sadness, and hope.

Automaton Creation

Users are able to create automatons: ancient, artificially-constructed robot-like creatures that follow their creator's commands to the fullest.

The highest level of Golems.

Alchemic Material Manipulation

User can create, shape and manipulate various alchemic materials. Through an esoteric usage of mystical and scientific techniques and ingredients.

Alchemic Constructs Manipulation

The user can change alchemic power (including their own bodies) into different forms and objects of varying detail and permanence, ranging from simple shapes and images, to individual elements, substances and materials.

Matter Transmutation

User can transmute matter into different forms. They are able to create and absorb matter and energy, shape, and rebuild it into anything of their choosing.

Herb Manipulation

The user can create, shape, and manipulate herbs, plants grown for medicinal value, for fragrance, or for flavoring food, and herbal remedies.

Manipulate the properties of herbs including the creation of mutant herbs.

Chemical Manipulation

The user can create, shape and manipulate chemicals, compounds or substances that have been purified or prepared, especially artificially.

Chemistry Manipulation

Users can manipulate chemistry, a discipline in science that is involved with elements and compounds composed of atoms, molecules and ions: their composition, structure, properties, behavior and the changes they undergo during a reaction with other substances.

Personal Alchemy

The user is capable of activating their own internalized alchemical processes.

Potion Manipulation

User can shape, create and manipulate potions, can have any kind of effect, and work when the person drinks or doused with it.

Pill Manipulation

User can shape, create and manipulate pills, can have any kind of effect, and work when the person consumes them.

Powder Manipulation

User can shape, create and manipulate powder, can have any kind of effect, and work when it enters the bloodstream.

Candy Manipulation

User can shape, create and manipulate candy, can have any kind of effect, and work when the person consumes them.

Arcane Focus of Darkness

Personal Darkness

The user can manipulate their own darkness and dark energy. Using this ability, one can effectively manipulate one’s own visibility by absorbing light, create a personalized field of shade or wield it for specialized techniques like attacks and defense.

Darkness Manipulation

Users can create, shape and manipulate darkness and shadows. Darkness is mostly used to cloud everything into total darkness, but can also be channeled to a variety of effects, both as an absence of light and a solid substance: one can also control and manipulate the beings that exist there, create and dispel shields and areas of total darkness, create constructs and weapons, teleport one's self through massive distances via shadows, etc.

Dark Energy Manipulation

The user can create, shape and manipulate dark energy, usually drawn from inter-dimensional or other similar sources. It can be channeled to a variety of effects as an absence of light, a solid, gaseous and/or liquid substance that can be shaped/manifested in various ways.

Shadow Marionette

The user takes control over motor functions of others against their will as if they were a marionette via shadows, subjects are still conscious, but can’t resist the user's puppet control. This can be executed by infusing one's shadow into the shadow of the target, physical contact, or simply having shadows touch.

Shadow Stealth

User can hide within the shadows and remain undetected.

Independent Shadow

The user has the ability to possess a sentient shadow. The shadow may only exist as a shadow or a physical manifestation of the user.

Dark Vision

Emit darkness/shadow from one's eyes.

Darkness/Shadow Telekinesis

The user can move, manipulate or otherwise interact with matter using darkness/shadows.

Personal Void

The user possesses a portable void and space within or around their bodies that they can use to entrap objects or living beings.

Darkness Attacks

The user can release/use darkness/shadow to attack in various shapes and/or intensities, either projected, used as a part of melee attacks, etc.

Here are some examples of darkness attack variations:

Dark Beam Emission: Release beams of darkness/shadow.

Dark Bolt Projection: Release many low powered projectiles of darkness/shadow.

Dark Pillar Projection: Project darkness/shadow pillars.

Darkness Blast: Release darkness/shadow over a specific target area.

Darkness Bomb Generation: Create bombs/explosions of darkness/shadow.

Darkness Bullet Projection: Fire in short sequence over a wide area.

... (and so on)

These are just a few examples, and there can be many more variations and techniques.
